---
title: token add
description: "Register a push token for a project. The token is stored locally; no server call is made"
---

## Usage

```bash
no-tickets token add [OPTIONS]
```

## Flags

| Flag | Short | Default | Description |
|---|---|---|---|
| `--force` |  |  | Overwrite an existing entry for the project |
| `--label <LABEL>` |  |  | Free-text label, surfaced by `nt status` / `nt token list` |
